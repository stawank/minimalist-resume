# Modern Resume Portfolio Template

A sleek, minimal, and responsive personal resume/portfolio landing page template built with vanilla HTML, CSS, and JavaScript. Perfect for developers, designers, and creative professionals looking to showcase their work and experience.

## Features

- 🌙 **Light/Dark Mode Toggle** - Smooth theme switching with persistent storage
- 📱 **Fully Responsive** - Optimized for all devices and screen sizes
- 🎨 **Modern Design** - Clean, minimal aesthetic with subtle animations
- 😊 **Emoji Integration** - Colorful emojis add personality while maintaining professionalism
- ⚡ **Vanilla Technologies** - No dependencies, just pure HTML, CSS, and JavaScript
- 🚀 **Easy Customization** - Simple to modify and personalize
- 📊 **Portfolio Sections** - Experience, projects, skills, education, and contact info

## Quick Start

1. Download or clone the repository
2. Open `index.html` in your browser
3. Customize the content with your own information
4. Deploy to your preferred hosting platform

## Customization

### Personal Information

Update the following sections in `index.html`:

- Header section: Name, title, location, and contact links
- About Me: Personal description
- Experience: Work history and achievements
- Projects: Portfolio projects with links
- Skills: Technical skills with emoji icons
- Education: Academic background

### Styling

The template uses CSS custom properties for easy theming. Key variables are defined in the `:root` selector:

- Colors and backgrounds
- Font sizes and spacing
- Shadow and transition effects

### Theme Colors

Both light and dark themes are included with smooth transitions. The theme preference is saved in localStorage and persists between sessions.
